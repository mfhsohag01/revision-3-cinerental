# 🎬 CineRental

A modern and responsive movie rental web application built with **React.js**. Users can browse a collection of movies, view detailed information in a modal, add or remove movies from the cart, calculate the total checkout price, and enjoy a smooth user experience with toast notifications and dark mode support.

# ⚙️ Getting Started

Clone the repository

```bash
git clone https://github.com/mfhsohag01/revision-3-cinerental.git
```

Go to the project directory

```bash
cd revision-3-cinerental
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
